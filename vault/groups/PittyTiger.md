## G0011

[PittyTiger](https://attack.mitre.org/groups/G0011) is a threat group believed to operate out of China that uses multiple different types of malware to maintain command and control.[^1] [^2] 

### Techniques Used
| ID | Name | Description |
| --- | --- | --- |
| [[Tool\|T1588.002]] | Tool | [PittyTiger](https://attack.mitre.org/groups/G0011) has obtained and used tools such as [Mimikatz](https://attack.mitre.org/software/S0002) and [gsecdump](https://attack.mitre.org/software/S0008).[^1]  |
| [[Valid Accounts\|T1078]] | Valid Accounts | [PittyTiger](https://attack.mitre.org/groups/G0011) attempts to obtain legitimate credentials during operations.[^1]  |
