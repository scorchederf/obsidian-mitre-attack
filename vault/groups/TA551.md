## G0127

[TA551](https://attack.mitre.org/groups/G0127) is a financially-motivated threat group that has been active since at least 2018. [^7]  The group has primarily targeted English, German, Italian, and Japanese speakers through email-based malware distribution campaigns. [^8] 

### Techniques Used
| ID | Name | Description |
| --- | --- | --- |
| [[Regsvr32\|T1218.010]] | Regsvr32 | [TA551](https://attack.mitre.org/groups/G0127) has used regsvr32.exe to load malicious DLLs.[^1]  |
| [[Email Addresses\|T1589.002]] | Email Addresses | [TA551](https://attack.mitre.org/groups/G0127) has used spoofed company emails that were acquired from email clients on previously infected hosts to target other individuals.[^8]  |
| [[Malicious File\|T1204.002]] | Malicious File | [TA551](https://attack.mitre.org/groups/G0127) has prompted users to enable macros within spearphishing attachments to install malware.[^8]  |
| [[Rundll32\|T1218.011]] | Rundll32 | [TA551](https://attack.mitre.org/groups/G0127) has used rundll32.exe to load malicious DLLs.[^8]  |
| [[Steganography\|T1027.003]] | Steganography | [TA551](https://attack.mitre.org/groups/G0127) has hidden encoded data for malware DLLs in a PNG.[^8]  |
| [[Spearphishing Attachment\|T1566.001]] | Spearphishing Attachment | [TA551](https://attack.mitre.org/groups/G0127) has sent spearphishing attachments with password protected ZIP files.[^1] [^8] [^7]  |
| [[Standard Encoding\|T1132.001]] | Standard Encoding | [TA551](https://attack.mitre.org/groups/G0127) has used encoded ASCII text for initial C2 communications.[^1]  |
| [[Domain Generation Algorithms\|T1568.002]] | Domain Generation Algorithms | [TA551](https://attack.mitre.org/groups/G0127) has used a DGA to generate URLs from executed macros.[^8] [^7]  |
| [[Command Obfuscation\|T1027.010]] | Command Obfuscation | [TA551](https://attack.mitre.org/groups/G0127) has used obfuscated variable names in a JavaScript configuration file.[^1]  |
| [[Web Protocols\|T1071.001]] | Web Protocols | [TA551](https://attack.mitre.org/groups/G0127) has used HTTP for C2 communications.[^1]  |
| [[Ingress Tool Transfer\|T1105]] | Ingress Tool Transfer | [TA551](https://attack.mitre.org/groups/G0127) has retrieved DLLs and installer binaries for malware execution from C2.[^8]  |
| [[Mshta\|T1218.005]] | Mshta | [TA551](https://attack.mitre.org/groups/G0127) has used mshta.exe to execute malicious payloads.[^8]  |
| [[Windows Command Shell\|T1059.003]] | Windows Command Shell | [TA551](https://attack.mitre.org/groups/G0127) has used `cmd.exe` to execute commands.[^8]  |
| [[Masquerading\|T1036]] | Masquerading | [TA551](https://attack.mitre.org/groups/G0127) has masked malware DLLs as dat and jpg files.[^8]  |
