## M1052

User Account Control (UAC) is a security feature in Microsoft Windows that prevents unauthorized changes to the operating system. UAC prompts users to confirm or provide administrator credentials when an action requires elevated privileges. Proper configuration of UAC reduces the risk of privilege escalation attacks. This mitigation can be implemented through the following measures:<br><br>Enable UAC Globally:<br><br>- Ensure UAC is enabled through Group Policy by setting `User Account Control: Run all administrators in Admin Approval Mode` to `Enabled`.<br><br>Require Credential Prompt:<br><br>- Use Group Policy to configure UAC to prompt for administrative credentials instead of just confirmation (`User Account Control: Behavior of the elevation prompt`).<br><br>Restrict Built-in Administrator Account:<br><br>Set `Admin Approval Mode` for the built-in Administrator account to `Enabled` in Group Policy.<br><br>Secure the UAC Prompt:<br><br>- Configure UAC prompts to display on the secure desktop (`User Account Control: Switch to the secure desktop when prompting for elevation`).<br><br>Prevent UAC Bypass:<br><br>- Block untrusted applications from triggering UAC prompts by configuring `User Account Control: Only elevate executables that are signed and validated`.<br>- Use EDR tools to detect and block known UAC bypass techniques.<br><br>Monitor UAC-Related Events:<br><br>- Use Windows Event Viewer to monitor for event ID 4688 (process creation) and look for suspicious processes attempting to invoke UAC elevation.<br><br>*Tools for Implementation*<br><br>Built-in Windows Tools:<br><br>- Group Policy Editor: Configure UAC settings centrally for enterprise environments.<br>- Registry Editor: Modify UAC-related settings directly, such as `EnableLUA` and `ConsentPromptBehaviorAdmin`.<br><br>Endpoint Security Solutions:<br><br>- Microsoft Defender for Endpoint: Detects and blocks UAC bypass techniques.<br>- Sysmon: Logs process creations and monitors UAC elevation attempts for suspicious activity.<br><br>Third-Party Security Tools:<br><br>- Process Monitor (Sysinternals): Tracks real-time processes interacting with UAC.<br>- EventSentry: Monitors Windows Event Logs for UAC-related alerts.
