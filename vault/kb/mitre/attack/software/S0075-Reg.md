## Description

[[kb/mitre/attack/software/S0075-Reg\|Reg]] is a Windows utility used to interact with the Windows Registry. It can be used at the command-line interface to query, add, modify, and remove information. [^1] <br><br>Utilities such as [[kb/mitre/attack/software/S0075-Reg\|Reg]] are known to be used by persistent threats. [^3]
