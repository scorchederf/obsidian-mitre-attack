## Description

The [[kb/mitre/attack/software/S0039-Net\|Net]] utility is a component of the Windows operating system. It is used in command-line operations for control of users, groups, services, and network connections. [^1] <br><br>[[kb/mitre/attack/software/S0039-Net\|Net]] has a great deal of functionality, [^6]  much of which is useful for an adversary, such as gathering system and network information for Discovery, moving laterally through [[kb/mitre/attack/techniques/T1021.002-SMB_Windows_Admin_Shares\|SMB/Windows Admin Shares]] using `net use` commands, and interacting with services. The net1.exe utility is executed for certain functionality when net.exe is run and can be used directly in commands such as `net1 user`.
