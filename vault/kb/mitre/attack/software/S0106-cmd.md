## Description

[[kb/mitre/attack/software/S0106-cmd\|cmd]] is the Windows command-line interpreter that can be used to interact with systems and execute other processes and utilities. [^2] <br><br>Cmd.exe contains native functionality to perform many operations to interact with the system, including listing files in a directory (e.g., `dir` [^4] ), deleting files (e.g., `del` [^3] ), and copying files (e.g., `copy` [^1] ).
