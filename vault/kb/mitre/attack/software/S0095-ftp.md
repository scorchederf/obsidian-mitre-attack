## Description

[[kb/mitre/attack/software/S0095-ftp\|ftp]] is a utility commonly available with operating systems to transfer information over the File Transfer Protocol (FTP). Adversaries can use it to transfer other tools onto a system or to exfiltrate data.[^2] [^1]
