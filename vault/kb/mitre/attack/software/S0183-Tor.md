## Description

[[kb/mitre/attack/software/S0183-Tor\|Tor]] is a software suite and network that provides increased anonymity on the Internet. It creates a multi-hop proxy network and utilizes multilayer encryption to protect both the message and routing information. [[kb/mitre/attack/software/S0183-Tor\|Tor]] utilizes "Onion Routing," in which messages are encrypted with multiple layers of encryption; at each step in the proxy network, the topmost layer is decrypted and the contents forwarded on to the next node until it reaches its destination. [^1]
