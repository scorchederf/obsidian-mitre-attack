## Description

[[kb/mitre/attack/software/S1144-FRP\|FRP]], which stands for Fast Reverse Proxy, is an openly available tool that is capable of exposing a server located behind a firewall or Network Address Translation (NAT) to the Internet. [[kb/mitre/attack/software/S1144-FRP\|FRP]] can support multiple protocols including TCP, UDP, and HTTP(S) and has been abused by threat actors to proxy command and control communications.[^1] [^4] [^2] [^3]
