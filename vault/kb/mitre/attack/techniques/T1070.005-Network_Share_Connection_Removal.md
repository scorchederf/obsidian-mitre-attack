## Description

Adversaries may remove share connections that are no longer useful in order to clean up traces of their operation. Windows shared drive and [[kb/mitre/attack/techniques/T1021.002-SMB_Windows_Admin_Shares\|SMB/Windows Admin Shares]] connections can be removed when no longer needed. [[kb/mitre/attack/software/S0039-Net\|Net]] is an example utility that can be used to remove network share connections with the `net use \\system\share /delete` command. [^1]
