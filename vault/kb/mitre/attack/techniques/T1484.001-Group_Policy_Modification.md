## Description

Adversaries may modify Group Policy Objects (GPOs) to subvert the intended discretionary access controls for a domain, usually with the intention of escalating privileges on the domain. Group policy allows for centralized management of user and computer settings in Active Directory (AD). GPOs are containers for group policy settings made up of files stored within a predictable network path `\<DOMAIN>\SYSVOL\<DOMAIN>\Policies\`.[^5] [^2]  <br><br>Like other objects in AD, GPOs have access controls associated with them. By default all user accounts in the domain have permission to read GPOs. It is possible to delegate GPO access control permissions, e.g. write access, to specific users or groups in the domain.<br><br>Malicious GPO modifications can be used to implement many other malicious behaviors such as [[kb/mitre/attack/techniques/T1053-Scheduled_Task_Job\|Scheduled Task/Job]], [[kb/mitre/attack/techniques/T1685-Disable_or_Modify_Tools\|Disable or Modify Tools]], [[kb/mitre/attack/techniques/T1105-Ingress_Tool_Transfer\|Ingress Tool Transfer]], [[kb/mitre/attack/techniques/T1136-Create_Account\|Create Account]], [[kb/mitre/attack/techniques/T1569.002-Service_Execution\|Service Execution]],  and more.[^2] [^9] [^11] [^1] [^10]  Since GPOs can control so many user and machine settings in the AD environment, there are a great number of potential attacks that can stem from this GPO abuse.[^9] <br><br>For example, publicly available scripts such as `New-GPOImmediateTask` can be leveraged to automate the creation of a malicious [[kb/mitre/attack/techniques/T1053-Scheduled_Task_Job\|Scheduled Task/Job]] by modifying GPO settings, in this case modifying `&lt;GPO_PATH&gt;\Machine\Preferences\ScheduledTasks\ScheduledTasks.xml`.[^9] [^11]  In some cases an adversary might modify specific user rights like SeEnableDelegationPrivilege, set in `&lt;GPO_PATH&gt;\MACHINE\Microsoft\Windows NT\SecEdit\GptTmpl.inf`, to achieve a subtle AD backdoor with complete control of the domain because the user account under the adversary's control would then be able to modify GPOs.[^4]
