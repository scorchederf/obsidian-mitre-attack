## Description

Adversaries may abuse CMSTP to proxy execution of malicious code. The Microsoft Connection Manager Profile Installer (CMSTP.exe) is a command-line program used to install Connection Manager service profiles. [^4]  CMSTP.exe accepts an installation information file (INF) as a parameter and installs a service profile leveraged for remote access connections.<br><br>Adversaries may supply CMSTP.exe with INF files infected with malicious commands. [^6]  Similar to [[kb/mitre/attack/techniques/T1218.010-Regsvr32\|Regsvr32]] / ”Squiblydoo”, CMSTP.exe may be abused to load and execute DLLs [^2]   and/or COM scriptlets (SCT) from remote servers. [^1]  [^3]  [^5]  This execution may also bypass AppLocker and other application control defenses since CMSTP.exe is a legitimate binary that may be signed by Microsoft.<br><br>CMSTP.exe can also be abused to [[kb/mitre/attack/techniques/T1548.002-Bypass_User_Account_Control\|Bypass User Account Control]] and execute arbitrary commands from a malicious INF through an auto-elevated COM interface. [^2]  [^3]  [^5]

| ID | Name | Description |
| --- | --- | --- |
| [[kb/mitre/attack/mitigations/M1038-Execution_Prevention\|M1038]] | Execution Prevention | Consider using application control configured to block execution of CMSTP.exe if it is not required for a given system or network to prevent potential misuse by adversaries. |
| [[kb/mitre/attack/mitigations/M1042-Disable_or_Remove_Feature_or_Program\|M1042]] | Disable or Remove Feature or Program | CMSTP.exe may not be necessary within a given environment (unless using it for VPN connection installation). |
