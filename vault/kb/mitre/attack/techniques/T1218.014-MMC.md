## Description

Adversaries may abuse mmc.exe to proxy execution of malicious .msc files. Microsoft Management Console (MMC) is a binary that may be signed by Microsoft and is used in several ways in either its GUI or in a command prompt.[^2] [^3]  MMC can be used to create, open, and save custom consoles that contain administrative tools created by Microsoft, called snap-ins. These snap-ins may be used to manage Windows systems locally or remotely. MMC can also be used to open Microsoft created .msc files to manage system configuration.[^8] <br><br>For example, `mmc C:\Users\foo\admintools.msc /a` will open a custom, saved console msc file in author mode.[^2]  Another common example is `mmc gpedit.msc`, which will open the Group Policy Editor application window. <br><br>Adversaries may use MMC commands to perform malicious tasks. For example, `mmc wbadmin.msc delete catalog -quiet` deletes the backup catalog on the system (i.e. [[kb/mitre/attack/techniques/T1490-Inhibit_System_Recovery\|Inhibit System Recovery]]) without prompts to the user (Note: `wbadmin.msc` may only be present by default on Windows Server operating systems).[^7] [^4] <br><br>Adversaries may also abuse MMC to execute malicious .msc files. For example, adversaries may first create a malicious registry Class Identifier (CLSID) subkey, which uniquely identifies a [[kb/mitre/attack/techniques/T1559.001-Component_Object_Model\|Component Object Model]] class object.[^5]  Then, adversaries may create custom consoles with the “Link to Web Address” snap-in that is linked to the malicious CLSID subkey.[^6]  Once the .msc file is saved, adversaries may invoke the malicious CLSID payload with the following command: `mmc.exe -Embedding C:\path\to\test.msc`.[^1]

| --- | --- | --- |
| [[kb/mitre/attack/mitigations/M1038-Execution_Prevention\|M1038]] | Execution Prevention | Use application control configured to block execution of MMC if it is not required for a given system or network to prevent potential misuse by adversaries. |
| [[kb/mitre/attack/mitigations/M1042-Disable_or_Remove_Feature_or_Program\|M1042]] | Disable or Remove Feature or Program | MMC may not be necessary within a given environment since it is primarily used by system administrators, not regular users or clients.  |
