## Description

Adversaries may utilize polymorphic code (also known as metamorphic or mutating code) to evade detection. Polymorphic code is a type of software capable of changing its runtime footprint during code execution.[^2]  With each execution of the software, the code is mutated into a different version of itself that achieves the same purpose or objective as the original. This functionality enables the malware to evade traditional signature-based defenses, such as antivirus and antimalware tools.[^4]  <br>Other obfuscation techniques can be used in conjunction with polymorphic code to accomplish the intended effects, including using mutation engines to conduct actions such as [[kb/mitre/attack/techniques/T1027.002-Software_Packing\|Software Packing]], [[kb/mitre/attack/techniques/T1027.010-Command_Obfuscation\|Command Obfuscation]], or [[kb/mitre/attack/techniques/T1027.013-Encrypted_Encoded_File\|Encrypted/Encoded File]].[^1] [^3] <br>
