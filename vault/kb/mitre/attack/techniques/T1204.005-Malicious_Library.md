## Description

Adversaries may rely on a user installing a malicious library to facilitate execution. Threat actors may [[kb/mitre/attack/techniques/T1608.001-Upload_Malware\|Upload Malware]] to package managers such as NPM and PyPi, as well as to public code repositories such as GitHub. User may install libraries without realizing they are malicious, thus bypassing techniques that specifically achieve Initial Access. This can lead to the execution of malicious code, such as code that establishes persistence, steals data, or mines cryptocurrency.[^2] [^1] <br><br>In some cases, threat actors may compromise and backdoor existing popular libraries (i.e., [[kb/mitre/attack/techniques/T1195.001-Compromise_Software_Dependencies_and_Development_Tools\|Compromise Software Dependencies and Development Tools]]). Alternatively, they may create entirely new packages and leverage behaviors such as typosquatting to encourage users to install them.
