## Description

Adversaries may attempt to extract credential material from the Security Account Manager (SAM) database either through in-memory techniques or through the Windows Registry where the SAM database is stored. The SAM is a database file that contains local accounts for the host, typically those found with the `net user` command. Enumerating the SAM database requires SYSTEM level access.<br><br>A number of tools can be used to retrieve the SAM file through in-memory techniques:<br><br>* pwdumpx.exe<br>* [[kb/mitre/attack/software/S0008-gsecdump\|gsecdump]]<br>* [[kb/mitre/attack/software/S0002-Mimikatz\|Mimikatz]]<br>* secretsdump.py<br><br>Alternatively, the SAM can be extracted from the Registry with Reg:<br><br>* `reg save HKLM\sam sam`<br>* `reg save HKLM\system system`<br><br>Creddump7 can then be used to process the SAM database locally to retrieve hashes.[^1] <br><br>Notes: <br><br>* RID 500 account is the local, built-in administrator.<br>* RID 501 is the guest account.<br>* User accounts start with a RID of 1,000+.<br>

## Mitigations
| ID | Name | Description |
| --- | --- | --- |
| [[kb/mitre/attack/mitigations/M1017-User_Training\|M1017]] | User Training | Limit credential overlap across accounts and systems by training users and administrators not to use the same password for multiple accounts. |
| [[kb/mitre/attack/mitigations/M1026-Privileged_Account_Management\|M1026]] | Privileged Account Management | Do not put user or admin domain accounts in the local administrator groups across systems unless they are tightly controlled, as this is often equivalent to having a local administrator account with the same password on all systems. Follow best practices for design and administration of an enterprise network to limit privileged account use across administrative tiers. |
| [[kb/mitre/attack/mitigations/M1027-Password_Policies\|M1027]] | Password Policies | Ensure that local administrator accounts have complex, unique passwords across all systems on the network. |
| [[kb/mitre/attack/mitigations/M1028-Operating_System_Configuration\|M1028]] | Operating System Configuration | Consider disabling or restricting NTLM.[^6]  |
