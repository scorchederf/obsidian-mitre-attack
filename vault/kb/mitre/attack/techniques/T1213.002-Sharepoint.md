## Description

Adversaries may leverage the SharePoint repository as a source to mine valuable information. SharePoint will often contain useful information for an adversary to learn about the structure and functionality of the internal network and systems. For example, the following is a list of example information that may hold potential value to an adversary and may also be found on SharePoint:<br><br>* Policies, procedures, and standards<br>* Physical / logical network diagrams<br>* System architecture diagrams<br>* Technical system documentation<br>* Testing / development credentials (i.e., [[kb/mitre/attack/techniques/T1552-Unsecured_Credentials\|Unsecured Credentials]])<br>* Work / project schedules<br>* Source code snippets<br>* Links to network shares and other internal resources<br>
