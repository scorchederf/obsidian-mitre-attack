## Description

Adversaries may exfiltrate data to text storage sites instead of their primary command and control channel. Text storage sites, such as `pastebin[.]com`, are commonly used by developers to share code and other information.  <br><br>Text storage sites are often used to host malicious code for C2 communication (e.g., [[kb/mitre/attack/techniques/T1608-Stage_Capabilities\|Stage Capabilities]]), but adversaries may also use these sites to exfiltrate collected data. Furthermore, paid features and encryption options may allow adversaries to conceal and store data more securely.[^1] <br><br>**Note:** This is distinct from [[kb/mitre/attack/techniques/T1567.001-Exfiltration_to_Code_Repository\|Exfiltration to Code Repository]], which highlight access to code repositories via APIs.
