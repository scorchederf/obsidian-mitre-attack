## S0233

[MURKYTOP](https://attack.mitre.org/software/S0233) is a reconnaissance tool used by [Leviathan](https://attack.mitre.org/groups/G0065). [^1] 

### Techniques Used
| ID | Name | Use |
| --- | --- | --- |
| [[Remote System Discovery\|T1018]] | Remote System Discovery | [MURKYTOP](https://attack.mitre.org/software/S0233) has the capability to identify remote hosts on connected networks.[^1]  |
| [[System Information Discovery\|T1082]] | System Information Discovery | [MURKYTOP](https://attack.mitre.org/software/S0233) has the capability to retrieve information about the OS.[^1]  |
| [[File Deletion\|T1070.004]] | File Deletion | [MURKYTOP](https://attack.mitre.org/software/S0233) has the capability to delete local files.[^1]  |
| [[Windows Command Shell\|T1059.003]] | Windows Command Shell | [MURKYTOP](https://attack.mitre.org/software/S0233) uses the command-line interface.[^1]  |
| [[Permission Groups Discovery\|T1069]] | Permission Groups Discovery | [MURKYTOP](https://attack.mitre.org/software/S0233) has the capability to retrieve information about groups.[^1]  |
| [[At\|T1053.002]] | At | [MURKYTOP](https://attack.mitre.org/software/S0233) has the capability to schedule remote AT jobs.[^1]  |
| [[Network Share Discovery\|T1135]] | Network Share Discovery | [MURKYTOP](https://attack.mitre.org/software/S0233) has the capability to retrieve information about shares on remote hosts.[^1]  |
| [[Network Service Discovery\|T1046]] | Network Service Discovery | [MURKYTOP](https://attack.mitre.org/software/S0233) has the capability to scan for open ports on hosts in a connected network.[^1]  |
| [[Local Account\|T1087.001]] | Local Account | [MURKYTOP](https://attack.mitre.org/software/S0233) has the capability to retrieve information about users on remote hosts.[^1]  |
