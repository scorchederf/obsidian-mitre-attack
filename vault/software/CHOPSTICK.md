## S0023

[CHOPSTICK](https://attack.mitre.org/software/S0023) is a malware family of modular backdoors used by [APT28](https://attack.mitre.org/groups/G0007). It has been used since at least 2012 and is usually dropped on victims as second-stage malware, though it has been used as first-stage malware in several cases. It has both Windows and Linux variants. [^7]  [^9]  [^6]  [^12]  It is tracked separately from the [X-Agent for Android](https://attack.mitre.org/software/S0314).

| [[File and Directory Discovery\|T1083]] | File and Directory Discovery | An older version of [CHOPSTICK](https://attack.mitre.org/software/S0023) has a module that monitors all mounted volumes for files with the extensions .doc, .docx, .pgp, .gpg, .m2f, or .m2o.[^9]  |
| [[Fileless Storage\|T1027.011]] | Fileless Storage | [CHOPSTICK](https://attack.mitre.org/software/S0023) may store RC4 encrypted configuration information in the Windows Registry.[^7]  |

| [[Domain Generation Algorithms\|T1568.002]] | Domain Generation Algorithms | [CHOPSTICK](https://attack.mitre.org/software/S0023) can use a DGA for [Fallback Channels](https://attack.mitre.org/techniques/T1008), domains are generated by concatenating words from lists.[^8]  |
