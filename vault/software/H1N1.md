## S0132

[H1N1](https://attack.mitre.org/software/S0132) is a malware variant that has been distributed via a campaign using VBA macros to infect victims. Although it initially had only loader capabilities, it has evolved to include information-stealing functionality. [^1] 

### Techniques Used
| ID | Name | Use |
| --- | --- | --- |
| [[Credentials from Web Browsers\|T1555.003]] | Credentials from Web Browsers | [H1N1](https://attack.mitre.org/software/S0132) dumps usernames and passwords from Firefox, Internet Explorer, and Outlook.[^2]  |
| [[Taint Shared Content\|T1080]] | Taint Shared Content | [H1N1](https://attack.mitre.org/software/S0132) has functionality to copy itself to network shares.[^2]  |
| [[Inhibit System Recovery\|T1490]] | Inhibit System Recovery | [H1N1](https://attack.mitre.org/software/S0132) disable recovery options and deletes shadow copies from the victim.[^2]  |
| [[Obfuscated Files or Information\|T1027]] | Obfuscated Files or Information | [H1N1](https://attack.mitre.org/software/S0132) uses multiple techniques to obfuscate strings, including XOR.[^1]  |
| [[Windows Command Shell\|T1059.003]] | Windows Command Shell | [H1N1](https://attack.mitre.org/software/S0132) kills and disables services by using cmd.exe.[^2]  |
| [[Replication Through Removable Media\|T1091]] | Replication Through Removable Media | [H1N1](https://attack.mitre.org/software/S0132) has functionality to copy itself to removable media.[^2]  |
| [[Windows Host Firewall\|T1686.003]] | Windows Host Firewall | [H1N1](https://attack.mitre.org/software/S0132) kills and disables services for Windows Firewall.[^2]  |
| [[Bypass User Account Control\|T1548.002]] | Bypass User Account Control | [H1N1](https://attack.mitre.org/software/S0132) bypasses user access control by using a DLL hijacking vulnerability in the Windows Update Standalone Installer (wusa.exe).[^2]  |
| [[Software Packing\|T1027.002]] | Software Packing | [H1N1](https://attack.mitre.org/software/S0132) uses a custom packing algorithm.[^1]  |
| [[Ingress Tool Transfer\|T1105]] | Ingress Tool Transfer | [H1N1](https://attack.mitre.org/software/S0132) contains a command to download and execute a file from a remotely hosted URL using WinINet HTTP requests.[^2]  |
| [[Data Encoding\|T1132]] | Data Encoding | [H1N1](https://attack.mitre.org/software/S0132) obfuscates C2 traffic with an altered version of base64.[^2]  |
| [[Symmetric Cryptography\|T1573.001]] | Symmetric Cryptography | [H1N1](https://attack.mitre.org/software/S0132) encrypts C2 traffic using an RC4 key.[^2]  |
| [[Disable or Modify Tools\|T1685]] | Disable or Modify Tools | [H1N1](https://attack.mitre.org/software/S0132) kills and disables services for Windows Security Center, and Windows Defender.[^2]  |
